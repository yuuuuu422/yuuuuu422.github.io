## 前言

**java反序列化三要素：**

1. "kick-off" gadget ==> 寻找重写readobject的类
2. chain gadget        ==> 将"kick-off" gadget 和 "sink" gadget连接起来
3. "sink" gadget        ==> RCE触发点

## 环境准备

因为不同链对jdk版本要求有所不同，后续分析分别用jdk7和jdk8搭建springboot简易入口，这里提一下idea已经不支持jdk7快速配置springboot，需要手动导入包。

jdk8:

```java
package com.theoyu.springbootdemo.start;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.io.InputStream;
import java.io.ObjectInputStream;
import java.util.Base64;


@RestController
public class HelloApi {
    @PostMapping("evil")
    public void hello(@RequestParam(value = "data")String data) throws IOException, ClassNotFoundException {
        byte[] b = Base64.getDecoder().decode(data);
        InputStream inputStream =new ByteArrayInputStream(b);
        ObjectInputStream objectInputStream =  new ObjectInputStream(inputStream);
        objectInputStream.readObject();
    }
}
```

Jdk7:

```java
package com.theoyu.boot.controller;
import org.apache.tomcat.util.codec.binary.Base64;
import org.springframework.web.bind.annotation.*;
import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.io.InputStream;
import java.io.ObjectInputStream;

@RestController
public class HelloController {
    @PostMapping("evil")
    public void hello(@RequestParam(value = "data")String  data) throws IOException, ClassNotFoundException {
        byte[]b= Base64.decodeBase64(data);
        InputStream inputStream =new ByteArrayInputStream(b);
        ObjectInputStream objectInputStream =  new ObjectInputStream(inputStream);
        objectInputStream.readObject();
    }
}
```
